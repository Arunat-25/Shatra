# Game sounds

Shatra ships UI sounds from the [lichess-org/lila](https://github.com/lichess-org/lila) repository, **piano** set.

| Asset path | Author | License |
|------------|--------|---------|
| `frontend/public/sounds/piano/*.ogg` | [Enigmahack](https://github.com/Enigmahack) | AGPL-3.0-or-later |

See lila [COPYING.md](https://github.com/lichess-org/lila/blob/master/COPYING.md) (section «Exceptions (free)»).

The **standard** wooden click set in lila is **not** redistributed here (non-free in COPYING).

## Refreshing sound files

```bash
./scripts/copy-lichess-sounds.sh
```
